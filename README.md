## 📌 Overview

**Probability Simulator ** is an interactive web application built using Streamlit that simulates probability experiments like coin toss and dice roll.

It not only shows experimental results but also compares them with theoretical probability and uses a simple **AI model** to predict future probability trends.

---

## ✨ Features

* 🎯 Coin Toss Simulation
* 🎲 Dice Roll Simulation
* 📊 Frequency & Probability Calculation
* 📈 Graphical Visualization
* ⚖️ Theoretical vs Experimental Comparison
* 🤖 AI Prediction (Linear Regression)
* 📥 Download Results as CSV
* 🎨 Modern UI with animations

---

## 🧠 Concepts Used

## 🚀 Run Locally

### 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/probability-simulator.git
cd probability-simulator
```

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run App

```bash
streamlit run app.py
```

---

## 🌐 Deployment

This app is deployed using **Streamlit Cloud**.

To deploy your own:

1. Push code to GitHub
2. Go to https://share.streamlit.io
3. Select your repo
4. Deploy

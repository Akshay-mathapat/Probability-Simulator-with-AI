# 🎲 Probability Simulator App

A Streamlit-based web app to simulate probability experiments like coin toss and dice roll.

## Features
- Choose experiment (Coin/Dice)
- Select number of trials
- View frequency and probabilities
- Compare theoretical vs experimental results
- Visual charts
